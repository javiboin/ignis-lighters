**Ignis** es una plataforma de e-commerce frontend moderna y veloz enfocada a la venta de encendedores de bolsillo de alta gama.

## 🛠️ Tecnologías

Este proyecto está construido aprovechando las mejores tecnologías actuales de frontend:
- **[React](https://reactjs.org/)** (Librería principal UI)
- **[Vite](https://vitejs.dev/)** (Herramienta de compilación veloz)
- **[React Router DOM](https://reactrouter.com/)** (Navegación y ruteo interno)
- **CSS3** (Estilado puro moderno)

## ✨ Características

- 🎨 **Diseño Premium**: Interfaz elegante, unificada y con una estética que resalta la calidad de la marca.
- 📱 **Totalmente Responsivo**: Experiencia de usuario perfecta tanto en dispositivos móviles como en navegadores de escritorio.
- ⚡ **Rendimiento Óptimo**: Desarrollado y servido a través de Vite, garantizando recargas en caliente al instante y una navegación increíblemente rápida.
- 🛒 **Colección y Navegación Dinámica**: Sistema de enrutamiento integrado para una navegación suave entre páginas como Inicio, Colección, Premium y Nosotros.

## 🚀 Instalación y uso local

Para correr este proyecto en tu máquina de foma local, sigue estos sencillos pasos:

1. **Instala las dependencias necesarias**  
   Ubicado en la carpeta del repositorio, ejecuta:
   ```bash
   npm install
   ```

2. **Inicia el servidor de desarrollo**  
   Arranca el entorno de Vite:
   ```bash
   npm run dev
   ```

3. **¡Listo!**  
   Abre [http://localhost:5173](http://localhost:5173) en tu explorador y empieza a navegar por la tienda.
